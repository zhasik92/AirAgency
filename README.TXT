Моделирование данных:

Населенный пункт: Имя, расположение, список аэропортов

Аэропорт: имя, город, список рейсов

Самолёты: айди,тип, маршрут, вместимость, список рейсов

Рейсы: откуда, куда, время вылета, время посадки, колличество проданных билетов, самолет.

Билет: айди, пассажир, рейс, цена.

Пассажир: имэйл, ФИО, год рождения, гражданство, номер паспорта, список билетов,
